Design a modern dark themed website UI for a funny private auction project between friends.

The product being auctioned is a small black men's pouch bag (a handheld pouch, not a neck pouch). The entire design should humorously treat this pouch as if it is the most legendary luxury product ever created.

The tone of the website should feel dramatic, premium, and exaggerated but also playful and funny.

Color style:
Use a dark theme dominated by black and charcoal tones because the product is a black pouch.
Accent colors can be gold, soft white, or subtle neon highlights to make the bag feel important and luxurious.

Typography:
Bold modern headings
Clean readable body text
Large hero titles that feel dramatic and slightly comedic

Create a responsive desktop first layout.

The website should only contain TWO pages.

Page 1: Landing Page

Sections required:

Hero section
Large dramatic headline presenting the pouch like a luxury product
Example tone: “The Most Important Pouch You Will Ever Own”
Large product image of a black pouch in the center
Short humorous description
Primary button: Enter the Auction

Features section
Show funny exaggerated benefits of the pouch
Examples:
Carry baby diapers like a responsible warrior
Emergency storage for sanitary pads
Protect your random cables like family treasure
Store chewing gum, power bank, and emotional damage
Hide receipts you will never look at again

Specifications section
Simple product style layout
Material
Capacity
Weight
"Legendary Storage Potential"

Why this pouch deserves an auction
Short comedic story style explanation about why this bag is valuable among friends

Auction preview section
Countdown timer preview
Text saying auction duration is 30 minutes
Button: Join the Auction

Email entry section
Simple email input
Button: Request OTP
Minimal login design

Footer
Very minimal
Small humorous copyright line


Page 2: Auction Page

This page should feel like a live auction environment.

Top section
Large product image of the black pouch
Title: The Legendary Pouch Auction

Auction status area
Countdown timer showing remaining time
Current highest bid
Highest bidder (masked email like s***@gmail.com)

User status box
Show whether the user is winning or has been outbid
Show user's current highest bid

Bid control area
Large buttons for bidding increments:
+5 AED
+10 AED
+15 AED
+20 AED
+50 AED

Buttons should look bold and interactive.

Cooldown indicator
Small message area saying
“Please wait 5 seconds before placing another bid”

Activity section
Display the latest bid updates in a simple list style
Example:
Ali placed a bid of 45 AED
Omar placed a bid of 50 AED

End of auction state
When timer reaches zero show
Winner announcement section
“Winner of the Legendary Pouch”

Design style guidelines:

Minimal pages
Clean layout
Centered product focus
Lots of spacing
Premium dark aesthetic
Funny exaggerated product marketing
Mobile friendly layout consideration

Components to design:

Buttons
Bid buttons
Countdown timer
Email OTP form
Auction status card
Bid activity list
Winner banner

The UI should feel modern, clean, and simple enough to easily convert into a React frontend.
